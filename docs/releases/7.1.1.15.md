# Release 7.1.1.15

**Release:** `7.1.1.15`

Release `7.1.1.15` completes the managed L2TP/IPsec UDP/1701 takeover for a
distribution-provided `xl2tpd.service`.

## Scope

- Stops a surviving package-managed XL2TPD listener after the systemd unit has
  been explicitly disabled and stopped.
- Requires the listener executable to be `xl2tpd` and its exact process cgroup
  to contain `xl2tpd.service`.
- Revalidates executable and cgroup ownership immediately before signaling the
  process.
- Keeps the existing MegaVPN command-line ownership checks for deployment
  runtimes.
- Leaves processes from unrelated systemd units or unknown cgroups untouched
  and reports the UDP/1701 conflict fail-closed.
- Applies the same lifecycle behavior to deployment Cleanup, the next Apply and
  emergency services cleanup.

## Operator Action

Deploy `7.1.1.15` to the Control Plane and update the provider-node agent to
`7.1.1.15`. The agent update is required because process ownership and cleanup
run on the provider node.

For an inactive L2TP/IPsec deployment blocked by a surviving
`xl2tpd.service` listener:

1. Run deployment `Cleanup`.
2. Confirm the cleanup job succeeds and UDP/1701 is free.
3. Run `Apply`.

If UDP/1701 remains occupied, inspect the process evidence:

```bash
sudo ss -H -lunp | grep ':1701 '
sudo readlink -f /proc/<pid>/exe
sudo tr '\0' ' ' </proc/<pid>/cmdline
sudo cat /proc/<pid>/cgroup
sudo systemctl status xl2tpd --no-pager -l
```

Do not terminate the reported process unless its ownership is established.

## Compatibility

- No PostgreSQL migration is required.
- Control Plane API contracts are unchanged.
- Existing provider profiles, encrypted secrets and deployments are unchanged.
- Non-L2TP external provider gateways are unaffected.

## Validation

Regression tests cover exact `xl2tpd.service` cgroup ownership, revalidation
before signal, an unrelated systemd service listener and the existing
MegaVPN-managed runtime paths. Go tests, agent race tests, static checks,
documentation consistency and the release self-test pass for this release.
